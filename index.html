Matterport 3D to Text Adventure
Nik Kim, Daragh Bryne
project repo / full work flow video

This project brings Matterport 3D experience to text adventure format. Basically, Matterport generates a navigatable 3D environment out of photos and hosts it through the web. They also provide accessible APIs and SDK toolkits to interact with the web model. This project uses these toolkits to convert the Matterport 3D environment to the Text Adventure format! Sounds interesting! Let's make your own adventure outof Matterport 3D model.

A Walk with a Shooting Star: Conversation Based Journal Generation
Nik Kim
journal template / demo video

The high-level concept of this game is to write a conversation-based journal while a player walks and talks with a virtual dog “Shooting Star”. Many of us hard to keep a journal when trying to write it alone, but while playing this game, we can effortlessly write a journal based on the content of our chats and conversations with a Chat GPT powered agent.
